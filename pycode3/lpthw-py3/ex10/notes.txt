The escape sequences available in python are:

+----+-------------------+--------------------------------+-----------------------------------------------------------+
|    | Escape Sequence   | Description                    | Example                                                   |
+====+===================+================================+===========================================================+
|  0 | \newline          | Backslash and newline ignored  | print("line1 \line2 \line3")                              |
|    |                   |                                | RESULT: line1 line2 line3                                 |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  1 | \\                | Backslash (\)                  | print("\\")                                               |
|    |                   |                                | RESULT: \                                                 |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  2 | \'                | Single quote (')               | print('\'')                                               |
|    |                   |                                | RESULT: '                                                 |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  3 | \"                | Double quote (")               | print("\"")                                               |
|    |                   |                                | RESULT: "                                                 |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  4 | \a                | ASCII Bell (BEL)               | print("\a")                                               |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  5 | \b                | ASCII Backspace (BS)           | print("Hello \b World!")                                  |
|    |                   |                                | RESULT: Hello World!                                    |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  6 | \f                | ASCII Formfeed (FF)            | print("Hello \f World!")                                  |
|    |                   |                                | RESULT: Hello                                             |
|    |                   |                                |          World!                                           |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  7 | \n                | ASCII Linefeed (LF)            | print("Hello \n World!")                                  |
|    |                   |                                | RESULT: Hello                                             |
|    |                   |                                |  World!                                                   |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  8 | \r                | ASCII Carriage Return (CR)     | print("Hello \r World!")                                  |
|    |                   |                                | RESULT:  World!                                           |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
|  9 | \t                | ASCII Horizontal Tab (TAB)     | print("Hello \t World!")                                  |
|    |                   |                                | RESULT: Hello    World!                                   |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
| 10 | \v                | ASCII Vertical Tab (VT)        | print("Hello \v World!")                                  |
|    |                   |                                | RESULT: Hello                                             |
|    |                   |                                |          World!                                           |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
| 11 | \ooo              | Character with octal value ooo | print("\110\145\154\154\157\40\127\157\162\154\144\41")   |
|    |                   |                                | RESULT: Hello World!                                      |
+----+-------------------+--------------------------------+-----------------------------------------------------------+
| 12 | \xhh              | Character with hex value hh    | print("\x48\x65\x6c\x6c\x6f\x20\x57\x6f\x72\x6c\x64\x21") |
|    |                   |                                | RESULT: Hello World!                                      |
+----+-------------------+--------------------------------+-----------------------------------------------------------+


Some escape sequences are only recognized in string literals. These are:
+----+-------------------+------------------------------------------------------------------------------------------+
|    | Escape Sequence   | Description                                                                              |
+====+===================+==========================================================================================+
|  0 | \N{name}          | Character named name in the Unicode database                                             |
+----+-------------------+------------------------------------------------------------------------------------------+
|  1 | \uxxxx            | Character with 16-bit hex value xxxx. Exactly four hexadecimal digits are required.      |
+----+-------------------+------------------------------------------------------------------------------------------+
|  2 | \Uxxxxxxxx        | Character with 32-bit hex value xxxxxxxx. Exactly eight hexadecimal digits are required. |
+----+-------------------+------------------------------------------------------------------------------------------+

! Hint: These beautiful ascii tables are created from csv using pandas and tabulate.
.. code: python

    import pandas
    from tabulate import tabulate

    data = pandas.read_csv("data_1.csv")
    print(tabulate(data, headers=data.columns, tablefmt="grid"))
